# Free Tool Strategy (Engineering as Marketing)

## Concept

Build useful free tools that:
1. Attract your target audience via SEO/sharing
2. Demonstrate your expertise
3. Capture leads or build brand awareness
4. Create backlink opportunities

## Tool Types

### Calculators
- ROI calculators
- Cost comparison tools
- Savings estimators
- Sizing/fit calculators

### Generators
- Template generators
- Policy generators (privacy, terms)
- Name/idea generators
- Code snippet generators

### Analyzers
- Website auditors (SEO, speed)
- Readability checkers
- Security scanners
- Performance graders

### Testers
- Email subject line testers
- Headline analyzers
- A/B test calculators
- Compatibility checkers

### Libraries/Resources
- Icon sets
- Color palettes
- Template collections
- Checklists

## Ideation Framework

### Find Tool Ideas
1. **Customer pain points** - What do they repeatedly ask?
2. **Manual processes** - What takes them time?
3. **Search demand** - What are they googling?
4. **Competitor gaps** - What tools don't exist?

### Validation Checklist
- [ ] Search volume > 1,000/month for related keywords
- [ ] No dominant free alternative
- [ ] Can build MVP in < 2 weeks
- [ ] Attracts your target buyer
- [ ] Natural connection to your product

## Lead Capture Strategies

### Ungated (Recommended Start)
- Full tool access, optional email for extras
- Build goodwill, maximize usage
- Capture leads via value-add (save results, extended report)

### Partially Gated
- Basic result free
- Detailed report requires email
- Balance reach vs lead capture

### Fully Gated
- Email required to use
- Higher quality leads, lower volume
- Risk: high bounce rate

## SEO Strategy

### Keyword Targeting
- "[thing] calculator"
- "[thing] generator"
- "free [tool type]"
- "online [tool type]"

### Content Pairing
- Blog post explaining the topic
- How-to guide using the tool
- Case studies with results

### Link Building
- Resource page outreach
- "Best [tool type]" list inclusion
- HARO/press mentions

## Technical Considerations

### Build vs Buy vs Embed
| Approach | Pros | Cons |
|----------|------|------|
| Custom build | Full control, branding | Dev time |
| No-code (Calculoid, Outgrow) | Fast, easy | Limited, subscription |
| Embed existing | Instant | Less differentiation |

### MVP Scope
- Core calculation/generation only
- Clean, simple UI
- Mobile-friendly
- Fast (< 2s result)
- Basic analytics

## Success Metrics

| Metric | Target |
|--------|--------|

| Avg time on page | > 2 min |
| Lead capture rate | 5-15% |
| Backlinks generated | 10+ |
| Search rankings | Page 1 for target keywords |
